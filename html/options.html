Useful when sharing the Merge Request on e.g instant messaging softwares
Available placeholders: {MR_TITLE}, {MR_ID}, {MR_URL}, {MR_DIFFS_URL}, {MR_AUTHOR_NAME}, {MR_STATUS}, {MR_SOURCE_BRANCH_NAME}, {MR_TARGET_BRANCH_NAME}, {MR_JIRA_TICKET_ID}, {MR_JIRA_TICKET_URL}
The ticket ID is automatically searched for in the source branch name and in the Merge Request title. The position of the ticket ID in these locations doesn't matter. Only the first uppercase ticket ID occurence will be matched.
